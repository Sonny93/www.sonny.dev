# Portfolio

Personal portfolio and blog — [www.sonny.dev](https://www.sonny.dev).

## Stack

- [Astro](https://astro.build) (static output, i18n routing `fr`/`en`, content collections)
- [UnoCSS](https://unocss.dev) for styling
- [three.js](https://threejs.org) animated space background

## Commands

| Command        | Action                          |
| -------------- | ------------------------------- |
| `pnpm dev`     | Start local dev server          |
| `pnpm build`   | Build static site to `dist/`    |
| `pnpm preview` | Preview the production build    |
| `pnpm check`   | Lint + format check + typecheck |

## Deploy

Deployed to GitHub Pages on every push to `main` — see [DEPLOY.md](./DEPLOY.md).
